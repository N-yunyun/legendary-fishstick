～最初に～

<!DOCTYPE html>

これはhtmlを使うという宣言

(この宣言は必ず1行目にしなければならない)jｍ

そのすぐ下に必ずhtmlタグを書く

<html>htmlを書き込む範囲</html>

この一行を<head>タグの中に書き込むことでどの端末で見ても同じような画面の表示にできる(この文はページごとにいちいち書き込まないといけないので、それぞれのHTMLファイルすべてに書き込まなければならない)
			↓
<meta name="viewport" content="width=device-width,initial-scale=1">

ブラケットの機能で、Ctrl + /でコメントアウトができる

終了タグあり= Y
終了タグなし= N

～その他タグ説明～

h1～h6タグ：Y

・見出しという大きな文字の見出しに使う

・h1(文字サイズ大)～h6(文字サイズ小)
　数字が大きくなるにつれて文字サイズも小さくなる

使い方：<h1>見出しの文章</h1>


pタグ：Y

・段落として区分けすることができる

使い方：<p>pタグでここが一つの段落として分けられる</p>


brタグ：N

・改行

・あまり使わない

使い方：あいうえお<br>かきくけこ

aタグ：Y

・リンクを作成

[属性]

・href = "値"

値に表示したいWebサイトのURLを指定

・target = "値(_self,_blankなど)"

_blankを指定すると表示したいWebサイトを別タブで開くことができる


使い方：

<a href = "URLを書く">表示する文章(なんのURLかなどの説明)</a>


imgタグ：Y

・画像を表示

[属性]

・src = "値"

値に表示したい画像のURLを指定

・alt = "値"

値に画像の説明を書く

・width = "値"

値に画像の表示サイズ(横幅)

・height = "値"

値に画像の表示サイズ(高さ)


使い方：<img src ="画像のURL"　alt="画像の説明">

コメントタグ：Y

・コメントを書き込むことができる

使い方：<!--　ここにコメントの内容を書く -->

文章を<!--と-->で囲む
