[概要]

本作品は分割画面でキャラクターを切り替えながら、変化というシステムをうまく使って敵に見つからないように進み、各部屋でパスワードの情報を入手し、最終的に目的の囚われた仲間のところまでたどり着けばゲームクリアです

[操作方法]

Xboxコントローラーを使ってください

左スティック：移動

Aボタン：決定・調べる・会話送り

Bボタン：キャンセル・変化解除

Xボタン,Yボタン：覚える・変化
　　　　 長押し：覚えた物を忘れる

LB,RBボタン:キャラクター切り替え

MENUボタン：ポーズ

[バグ]

人間側の会話ウィンドウがミニマップに隠れて見えなくなっています。
その他、特定の状態でプレイヤーの操作を切り替えたときにバグが発生します。

[作成部分]

プレイヤー挙動を一部担当、UI素材の制作、企画書の制作、会話システムの制作

[システム]

<キャラクター>

人間：

敵に変装(変化)し、変装した状態で動き回ったり、敵に話しかけたりすることができる。

変化できる時間に限りがある

閉まっている扉の中に入ったり、パスワードの入力することができる

犬：

物に変化し、変化した状態で動くことができないが、狭い通気口などを通ったりして別の部屋に行くことができる

変化できる時間は無制限

二つのキャラクターの共通点：

左上にある残機と警戒度は二人で共有しているため、どちらかが捕まったり、サーチライトにあたっても同じように減る

最初から空いている扉に入ることができる


<変化>

覚えることができる物や人に近づくと、Xボタンや青いカーソルが表示されます。

物や人を覚えると、左上のアイコンに覚えた物や人が表示されます。

そこに書いてあるボタンを押せば、そのボタンに連動している「覚えた物や人」に変化することができます。

連動しているボタンを長押しすると覚えたものを忘れることができます。

仕様：

変化した状態では敵に捕まらない
覚えられる物や人は人間と犬でそれぞれ最大で2つまで

<ギミック>

道中に近づくとAボタンが表示されるオブジェクトがあります。

その指示に従い、Aボタンを押すことでそのギミックを解除したり、敵の会話を盗み聞きすることができます。

犬にしか押せないスイッチと人間にしか押せないスイッチがあります

行き先をふさぐゲートを解除するには、キャラクターを切り替えてそのゲートに対応するスイッチを押してきてもらう必要があります。
