[概要]
上からオブジェクトを落として合体(進化)させていき、なるべく多くのオブジェクトを作ってハイスコアを目指すスイカゲームライクの2D落ちものマージゲームです。

[操作方法]
十字キー　←→　移動
スペースキー　オブジェクトを落とす

[こだわりポイント]
オブジェクトのステータス情報をスクリプタブルオブジェクトでデータファイルにし、メモリの削減とデータの追加にも柔軟に対応できるようにしました。
ゲームオブジェクトごとにクラスを作り、オブジェクト指向を意識してクラス分けを行い、メソッドもできる限り細かく分けました。
オブジェクトが進化するときに、自然に進化したように見えるよう、線形補完なども行っています。
