[概要]
スマートフォン向けに開発している3Dカーレースゲームです。デバッグ用程度にWindowsでもプレイ可能にしています。
ステージ各地に設置されたアーチをすべてくぐり、最終的にゴールアーチをくぐるとゲームクリアです。(現在はゴールアーチのみに判定あり)
今後はタイム計測システムやSE,BGMなど、車種変更システムなどを追加していく予定です。

[こだわりポイント]

クラスを細かく分け、『他から値を受け取り、計算のみを行うクラス』、『車の動きのみを担当するクラス』、『Uiのボタンを管理するクラス』、『プレイヤーがAndroidを使用している場合に使用するプレイヤークラス』、『プレイヤーがWindowsを使用している場合に使用するプレイヤークラス』といった感じで設計しています。
Android向けなので、できるだけ処理が重くならないように、オクルージョンカーリングや早期リターンを行い、Updateを使用せず、UIのボタンで入力を受け取る方式を採用しています。
できるだけ処理を重くせず、かつ車の実際の重さやスピード感を少しでもリアルに再現するため、リジッドボディとフィジックマテリアルの値を試行錯誤し、調整しています。

[操作方法]

Menu　　　　   ポーズ画面を開く
Auto　　　　　 一度押すと自動でアクセルを踏む(もう一度押すとオートアクセル解除)
accelerator　　  押している間アクセルを踏む
reverse　　　　押している間バックする
Resume Game　ゲームに戻る
Return Title　　タイトルに戻る

Aキー　　　　　左カーブ
Dキー　　　　　右カーブ
